# context-mode-go

A 100% NPM/NodeJS-free, Go implementation of Mert Koseoglu's [context-mode](https://github.com/mksglu/context-mode).

This tool runs as a local-first Model Context Protocol (MCP) server that acts as a virtualization layer for tool outputs, allowing AI coding agents to execute heavy tasks and save up to 98% in token usage.
